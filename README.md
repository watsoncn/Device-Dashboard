# Device Dashboard

### To run

* Fork or clone the project:

```
git clone https://github.com/watsoncn/Device-Dashboard.git
```

* Then install the dependencies:

```
npm install
```

* Run development server:

```
npm start
```

Open the web browser to `http://localhost:8888/`

### To test

To run unit tests:

```
npm test
```
